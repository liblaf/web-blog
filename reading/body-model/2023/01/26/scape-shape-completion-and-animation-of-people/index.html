                              # 4. Pose Deformation

                              # 4.1. Deformation Process

                              vk,ji=Rℓ[k]iQkiv^k,j,j=2,3
                              • ( input ) Yi — mesh in the data set containing different poses of a human
                              • ( input ) pk — triangle of the template
                              • ( input ) xk,1,xk,2,xk,3 — points of triangle pk
                              • ( input ) v^k,ji=xk,j−xk,1,j=2,3 — triangle edges
                              • ( params ) Qki — 3×3 linear transformation matrix
                              • ( input ) Rℓi — rotation of the polygon’s rigid part in the articulated skeleton
                              • ( input ) ℓ[k] — body part associated with triangle pk

                              However, the predictions for the edges in different triangles are rarely consistent. Thus, to construct a single coherent mesh, we solve for the location of the points y1,…,yM that minimize the overall least squares error:

                              argminy1,…,yM⁡∑k∑j=2,3‖Rℓ[k]iQkiv^j,k−(yj,k−y1,k)‖2

                              # 4.2. Learning the Pose Deformation Model

                              t=‖θ‖2sin⁡‖θ‖[m32−m23m13−m31m21−m12]withθ=cos−1⁡(tr⁡(M)−12)
                              • ( analogous to Qki ) M — any 3×3 rotation matrix
                              • mij — M’s i-th row and j-th column
                              • t — twist for the joint angle, a 3D vector
                                • direction of t — axis of rotation
                                • magnitude of t — rotation amount

                              predicts the transformation matrices Qki as a function of the twists of its two nearest joints Δrℓ[k]i=(Δrℓ[k],1i,Δrℓ[k],2i)

                              qk,lmi=ak,lmT⋅[Δrℓ[k]i1]l,m=1,2,3
                              • ( params ) ak,lm — 7×1 regression vector
                              Qki=Qak(Δrℓ[k]i)argminak,lm⁡∑i([Δri1]ak,lm−qk,lmi)2argmin{Q1i,…,QPi}⁡∑k∑j=2,3‖RkiQkiv^k,j−vk,j‖2+ws∑k1,k2 adjI(ℓk1=ℓk2)⋅‖Qk1i−Qk2i‖2
                              • ( input ) ws=0.001ρ
                              • ( input ) ρ — resolution of the model mesh X
                              • I(⋅) — indicator function
