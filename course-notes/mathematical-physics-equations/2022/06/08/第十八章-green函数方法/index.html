                              # 稳定问题的 Green 函数

                              # 对于 Poisson 方程的定解问题

                              (1a)∇2⁡u=ρ,r∈V,(1b)(αu+β∂u∂n)Σ=f(Σ),

                              其 Green 函数 G(r;r′) 是点源问题

                              (2a)∇2⁡G(r;r′)=−δ(r−r′),r,r′∈V

                              在齐次边界条件

                              (2b)(αG+β∂G∂n)Σ=0

                              下的解. 这里需约定 α≠0. 当 α=0 (第二类边界条件) 时, 需另行定义广义 Green 函数.

                              # 对于 Helmholtz 方程的定解问题

                              (3a)∇2⁡u+k2u=ρ,r∈V,(3b)(αu+β∂u∂n)Σ=f(Σ),

                              其 Green 函数 G(r;r′) 也是非齐次偏微分方程

                              (4a)∇2⁡G(r;r′)+k2G(r;r′)=−δ(r−r′),r,r′∈V

                              在齐次边界条件

                              (4b)(αG+β∂G∂n)Σ=0

                              下的解.

                              # G(r;r′) 的奇异性

                              r=r′ 是 G(r;r′) 的奇点, 但具体的奇异行为随空间维数而异.

                              # G(r;r′) 的对称性

                              当 G(r;r′) 满足定解问题 (4a), (4b) 时, 则具有对称性:

                              G(r;r′)=G(r′;r).

                              对于定解问题 (1a), (1b) 或 (3a), (3b), 均有解

                              (5)u(r)=−∭VG(r;r′)ρ(r′)dr′−1α∬Σf(Σ′)∂G(r;r′)∂n′∫|Σ′dΣ′,

                              其中 G(r;r′) 是定解问题 (2a), (2b) 或 (4a), (4b) 的解.

                              # 波动方程的 Green 函数

                              以一维有界区间内的波动方程定解问题

                              (6a)∂2u∂t2−a2∂2u∂x2=f(x,t),0<x<l, t>0,(6b)u∫|x=0=μ(t),u∫|x=l=ν(t),t>0,(6c)u∫|t=0=ϕ(x),∂u∂t∫|t=0=ψ(x),0<x<l

                              为例, 其 Green 函数 G(x,t;x′,t′) 是瞬时 (仅存在于某一时刻) 点 (仅存在于空间某点) 源问题

                              (7a)(∂2∂t2−a2∂2∂x2)G(x,t;x′,t′)=δ(x−x′)δ(t−t′),x<x,x′<l, t,t′>0

                              在齐次定解条件

                              (7b)G(x,t;x′,t′)∫|x=0=0,G(x,t;x′,t′)∫|x=l=0,t,t′>0,(7c)G(x,t;x′,t′)∫|t<t′=0,∂G(x,t;x′,t′)∂t∫|t<t′=0,0<x,x′<l

                              下的解. 去掉定解问题 (7a), (7b), (7c) 中的限制条件 t,t′>0, 也就是将 Green 函数 G(x,t;x′,t′) 的定义域延拓为 −∞<t,t′<∞, 则 G(x,t;x′,t′) 具有空间变量的对称性和时间变量的倒易性

                              G(x,t;x′,t′)=G(x′,−t′;x,−t).

                              可以用此 Green 函数 G(x,t;x′,t′) 表示出定解问题 (7a), (7b), (7c) 的解

                              u(x,t)=∫0ldx∫0tG(x,t;x′,t′)f(x′,t′)dt′+∫0l[G(x,t;x′,0)ψ(x′)−ϕ(x′)∂G(x,t;x′,t′)∂t′∫|t′=0]dx′(8)−a2∫0t[ν(t′)∂G(x,t;x′,t′)∂x′∫|x′=l−μ(t′)∂G(x,t;x′,t′)∂x′∫|x′=0]dt′.

                              热传导问题也可类似地讨论.

                              # Green 函数的求法

                              Green 函数满足的是非齐次方程、齐次定解条件的定解问题, 因此可以用按相应齐次问题本征函数展开法求解, 也可用积分变换法求解; 另一方面, 由于出现的是特殊的非齐次项 (点源或瞬时点源, 数学形式为 δ 函数), 故又有一些特殊的解法, 如电像法.
