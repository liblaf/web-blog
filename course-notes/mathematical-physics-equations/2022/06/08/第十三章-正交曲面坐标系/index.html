                                              # 正交曲面坐标系中的问分算符表达式

                                              设正交曲面坐标系的三个坐标为 ξ1, ξ2, ξ3, 其孤元 ds 为

                                              ds2=∑i=13Hi2dξi2,

                                              其中 Hi 是此正交曲面坐标系的度规, 一般是 ξ1, ξ2, ξ3 的函数. 它可以通过直角坐标与正交曲面坐标之间的关系算出

                                              Hi2=(∂x∂ξi)2+(∂y∂ξi)2+(∂z∂ξi)2,i=1,2,3.

                                              在球坐标系中,

                                              ξ1=r,ξ2=θ,ξ3=ϕ,ds1=dr,ds2=rdθ,ds3=rsin⁡θdϕ,H1=1,H2=r,H3=rsin⁡θ.

                                              在柱坐标系中,

                                              ξ1=r,ξ2=ϕ,ξ3=z,ds1=dr,ds2=rdϕ,ds3=dz,H1=1,H2=r,H3=1.

                                              # 标量函数 u 的梯度

                                              # 一般正交曲面坐标系中标量函数 u(ξ1,ξ2,ξ3) 的梯度

                                              (1)∇⁡u=(1H1∂u∂ξ1,1H2∂u∂ξ2,1H3∂u∂ξ3).

                                              # 球坐标系中标量函数 u(r,θ,ϕ) 的梯度

                                              ∇⁡u=(∂u∂r,1r∂u∂θ,1rsin⁡θ∂u∂ϕ).

                                              # 柱坐标系中标量函数 u(r,ϕ,z) 的梯度

                                              ∇⁡u=(∂u∂r,1r∂u∂ϕ,∂u∂z).

                                              # 矢量函数 A 的散度

                                              # 一般正交曲面坐标系中矢量函数 A(ξ1,ξ2,ξ3) 的散度

                                              (2)∇⋅A=1H1H1H3∑i=13∂∂ξi(H1H2H3HiAi)

                                              # 球坐标系中适量函数 A(r,θ,ϕ) 的散度

                                              ∇⋅A=1r2∂∂r(r2Ar)+1rsin⁡θ∂∂θ(sin⁡θAθ)+1rsin⁡θ∂Aϕ∂ϕ.

                                              # 柱坐标系中适量函数 A(r,ϕ,z) 的散度

                                              ∇⋅A=1r∂∂r(rAr)+1r∂Aϕ∂ϕ+∂Az∂z.

                                              # Laplace 算符的不变性

                                              在常用的坐标变换下, 包括有线性变换 (如平移变换), 正交变换 (如空间转动) 及空间反射 (r′=−r) 等, Laplace 算符具有不变性,

                                              ∂2∂x′2+∂2∂y′2+∂2∂z′2=∂2∂x2+∂2∂y2+∂2∂z2.

                                              # Helmholtz 方程 ∇2⁡u+k2u=0 在球坐标和柱坐标中分离变量

                                              # 球坐标系中 Helmholtz 方程为

                                              1r2∂∂r(r2∂u∂r)+1r2sin⁡θ∂∂θ(sin⁡θ∂u∂θ)+1r2sin2⁡θ∂2u∂ϕ2+k2u=0,

                                              令 u(r,θ,ϕ)=R(r)Θ(θ)Φ(ϕ), 则可分离出方程

                                              1r2ddr(r2dRdr)+(k2−μr2)R=0,1sin⁡θddθ(sin⁡θdΘdθ)+(μ−m2sin2⁡θ)Θ=0,Φ″+m2Φ=0.

                                              第一个方程可化为 Bessel 方程, 当 k=0 时, 它是 Euler 型方程, 第二个方程称为连带 Legendre 方程.

                                              # 柱坐标系中 Helmholtz 方程为

                                              1r∂∂r(r∂u∂r)+1r2∂2u∂ϕ2+∂2u∂z2+k2u=0.

                                              令 u(r,ϕ,z)=R(r)Φ(ϕ)Z(z), 则可分离出方程

                                              1rddr(rdRdr)+(k2−μ−m2r2)R=0,Φ″+m2Φ=0,Z″+μZ=0,

                                              第一个方程可化为 Bessel 方程, 当 k2−μ=0 时, 退化为 Euler 型方程.

                                              # 在圆内第一类边值问题 (圆内 Dirichlet 问题) 中

                                              由周期边界条件构成本征值问题

                                              Φ″(ϕ)+νΦ(ϕ)=0,ϕ∈(0,2π),Φ(0)=Φ(2π),Φ′(0)=Φ′(2π).

                                              对本征值 ν0=0, 本征函数

                                              Φ0(ϕ)=1;

                                              对本征值 νm=m2 (m≠0), 本征函数是二重简并的

                                              Φm(ϕ)={sin⁡mϕ,cos⁡mϕ,m=1,2,3,…

                                              或

                                              Φm(ϕ)=eimϕ,m=±1,±2,±3,….

                                              本征函数的这两种取法, 不仅不同本征值的本征函数相互正交, 而且两个简并本征函数之间也是正交的.
