                                              # 伴算符与自伴算符

                                              # 1.

                                              设 L^ 和 M^ 为定义在一定函数空间内的线性 (微分) 算符, 若对于该函数空间内的任意函数 u 和 v, 恒有

                                              (1)(v,L^u)=(M^v,u)即∫abv∗L^udx=∫ab(M^v)∗udx,

                                              则称 M^ 是 L^ 的 伴算符.

                                              反之, L^ 也是 M^ 的伴算符.

                                              # 2.

                                              若算符 L^ 的伴算符就是自身, 则称 L^ 是 自伴算符. 这是有

                                              (2)(v,L^u)=(L^v,u)即∫abv∗L^udx=∫ab(L^v)∗udx.

                                              # 自伴算符本征值问题的基本性质

                                              前提是自伴算符的本粧值问题是否有解?

                                              需要区别正则的与奇异的两种情形: 如果算符的定义域是无界或半无界区间, 或者区间的端点是方程的奇点, 则此本征值问题是奇异的, 否则就属于正则的本征值问题.

                                              在可分的 Hilbert 空间内, 正则的自律算符本征值问题一定有解, 而且本征值是离散的 (因而构成可数集).

                                              对于奇异的自伴算符, 其本征值问题则不一定有解, 即使有解, 也可能是连续谱, 或者离散谱与连续谱二者兼而有之.

                                              在自伴算符本征值问题有解的前提下, 求得的本征值与本征函数具有下列性质:

                                              # (1)

                                              自伴算符的本征值必为实数.

                                              # (2)

                                              自伴算符的本征函数具有正交性, 即对应不同本征值的本征函数一定正交.

                                              # (3)

                                              自伴算符的本征函数 (的全体) 构成一个完备函数组, 即任意一个在区间 [a,b] 中有连续二阶导数, 旦满足和自伴算符 L^ 相同的边界条件的函数 f(x), 均可按本征函数 {yn(x)} 展开为绝对而且一致收敛的级数

                                              (3a)f(x)=∑n=1∞cnyn(x),

                                              其中

                                              (3b)cn=∫abf(x)yn∗(x)dx∫abyn(x)yn∗(x)dx.

                                              上述展开条件还可以放宽为: 对于在 [a,b] 中平方可积的任意函数, (3a) 在平均收敛

                                              (3c)limN→∞∫ab|f(x)−∑n=1Ncnyn(x)|2dx=0

                                              的意义下仍然成立, 其展开系数仍为 (3c).

                                              # Sturm – Liouville 型方程的本征值问题

                                              # 1.

                                              Sturm -– Liouville 型方程

                                              ddx[p(x)dydx]+[λρ(x)−q(x)]y=0,a<x<b

                                              在边界条件

                                              p(x)(y1∗dy2dx−y2dy1∗dx)∫|ab=0

                                              下构成自伴算符的本征值问题, 其中 λ 为参数, ρ(x) 为权函数.

                                              使此问题有非零解的 λ 值称为本征值, 相应的非零解称为本征函数 (以下相同).

                                              # 在常见的 Sturm – Liouville 型方程中, p(x), q(x) 和 ρ(x) 均满足以下条件:

                                              1. p(x)⩾0, 但不恒为 0, 并且只在边界点 (a, 或 b, 或 a 和 b) 可能为 0.
                                              2. ρ(x)⩾0, 但不恒为 0.
                                              3. q(x)⩾0, 且 q(x)/p(x) 在 [a,b] 中, 除 a, b 两点可能是不超过二阶的极点外, 是实的连续函数.

                                              # Sturm – Liouville 型方程本征值问题的基本类型

                                              # 与第一、二、三类边界条件构成的本征值问题

                                              ddx[p(x)dy(x)dx]+[λρ(x)−(x)]y(x)=0,x∈(a,b),α1y′(a)−β1y(a)=0,α2y′(b)+β2y(b)=0,

                                              其中 α1, β1, α2, β2 均为非负常数, 且 α1, β1 不同时为零, α2, β2 不同时为零.

                                              # 存在有界条件时的本征值问题

                                              例如,

                                              ddx[p(x)dy(x)dx]+[λρ(x)−q(x)]y(x),x∈(a,b),y(a) 有界,α2y′(b)+β2y(b)=0.

                                              出现此类本征值问题的前提条件是 p(a)=0, x=a 是方程的正则奇点, 且方程在 x=a 点有一个解是发散的, 这是需加入一个有界条件, 将此解剔除.

                                              若 p(b)=0 或 p(a)=p(b)=0, 可做类似处理.

                                              # 存在周期条件时的本征值问题

                                              ddx[p(x)dy(x)dx]+[λρ(x)−q(x)]y(x)=0,x∈(a,b),y(a)=y(b),y′(a)=y′(b).

                                              出现此本征值问题的条件是 p(a)=p(b), q(a)=q(b), ρ(a)=ρ(b).

                                              # 几种常见的 Sturm – Liouville 型方程

                                              # X″(x)+λX(x)=0 或 Φ″(ϕ)+μΦ(ϕ)=0

                                              在这类方程中, p(x)=1, q(x)=0, 权函数 ρ(x)=1, 两方程中的待定参数为 λ 或 μ.

                                              # Bessel 方程

                                              1rddr[rdR(r)dr]+(k2−m2r2)R(r)=0,

                                              在此方程中, p(r)=r, q(r)=m2r, 权函数 ρ(r)=r, 参数为 k2.

                                              # 球 Bessel 方程

                                              1r2ddr[r2dR(r)dr]+[k2−l(l+1)r2]R(r)=0,

                                              在此方程中, p(r)=r2, q(r)=l(l+1), 权函数 ρ(r)=r2, 参数为 k2.

                                              # 连带 Legendre 方程

                                              1sin⁡θddθ[sin⁡θdΘ(θ)dθ]+(λ−m2sin2⁡θ)Θ(θ)=0,

                                              在此方程中, p(θ)=sin⁡θ, q(θ)=m2sin⁡θ, 权函数 ρ(θ)=sin⁡θ, 参数为 λ.

                                              做变换 cos⁡θ=x, Θ(θ)=y(x) 可得到连带 Legendre 方程的另一个标准形式

                                              ddx[(1−x2)dydx]+(λ−m21−x2)=0,

                                              这时 p(x)=1−x2, q(x)=m21−x2, 权函数 ρ(x)=1, 参数仍为 λ.

                                              # Sturm – Liouville 型方程本征值问题的退化现象

                                              在边界条件

                                              p(x)(y1∗d2y2dx2−y2d2y1∗dx2)∫|a=p(x)(y1∗d2y2dx2−y2d2y1∗dx2)∫|b=0

                                              下, Sturm – Liouville 型方程

                                              ddx[p(x)dydx]+[λρ(x)−q(x)]=0,a<x<b

                                              的本征值问题是非退化的, 即对应于一个本征值, 只有一个本征函数; 只在周期条件下, 对应于一个本征值, 才可能有两个本征函数; 这两个本征函数可能正交, 也可能不正交, 但是通过正交化步骤, 总可以将对应于同一本征值的两个本征函数正交化.
